# DREVENANT7 - Pojavlauncher Expert Website

![DREVENANT7](profile.jpeg)

## Overview

This is the personal website of DREVENANT7, a leading expert in Pojavlauncher for running Minecraft Java Edition on mobile devices (iOS and Android). The site serves as a hub for tutorials, guides, and community resources related to Pojavlauncher, Optifine setup, shaders, and troubleshooting common issues.

## Features

- **Animated Background**: Dynamic particle system with canvas-based animations
- **Profile Section**: Rainbow-animated username with bio and avatar
- **Social Links**: Direct links to Telegram, TikTok, YouTube, and support page
- **Minecraft Mod Showcase**: Highlighting personal mod development
- **Responsive Design**: Optimized for desktop and mobile devices
- **Loading Animation**: Smooth loading overlay with progress bar
- **Interactive Elements**: Hover effects, music toggle, and dialogue system

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom animations, gradients, and responsive design
- **JavaScript**: Particle animations, loading progress, and interactive features
- **Canvas API**: Background particle effects

## Installation & Usage

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/drevenant7-website.git
   cd drevenant7-website
   ```

2. **Open in Browser**:
   - Simply open `index.html` in your web browser
   - No server required for static content

3. **Local Development**:
   - Use a local server for better development experience:
     ```bash
     python -m http.server 8000
     ```
   - Open `http://localhost:8000` in your browser

## Customization

- **Colors**: Modify gradients and colors in `style.css`

- **Content**: Update text, links, and images in `index.html`
- **Particles**: Configure particle system in `script.js`
